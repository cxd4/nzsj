This is the Latin-based character set for using English in naming the file. Code points for these characters range from 0xAB for the capital letter A all the way down to 0xDE for the lowercase letter z, not counting the final row of decimal digits and the period, dash, and space characters.

Translation of the buttons and remaining kana characters still on this screen are yet to be evaluated.

Notes on the Latin translation table:

  • (*) Extraneous in modern Japanese. No longer used in modern writing and not actually present in the game's character set but shown in this table merely as placeholders.
  • (**) Since wa is the only sound starting with a 'w' (wo is more traditionally expressed as just o), this column does not adhere to the a/i/u/e/o organizational structure of the rest of the table. The "n" sound, which does not belong to any of the sound columns in this table, is placed here under "wo", while "wo" maps to the game screen's position of where "wu" would be if such a syllable even existed.
